Automate your BioReactor

Automatice su Biorreactor

Learn how to schedule tasks to control your devices automatically. Perfect for circadian rhythm experiments or timed agitation.

Aprenda cómo programar tareas para controlar sus dispositivos automáticamente. Perfecto para experimentos de ritmo circadiano o agitación temporizada.

Important: Importante:
  • Always use the IP address displayed on the device's LCD screen. The examples below use placeholders like 192.168.1.50.
  • Use siempre la dirección IP que aparece en la pantalla LCD del dispositivo. Los ejemplos a continuación usan marcadores de posición como 192.168.1.50.
  • The devices only support 2.4GHz WiFi networks. Higher frequencies (5GHz, 6GHz) are not supported.
  • Los dispositivos solo soportan redes WiFi de 2.4GHz. Las frecuencias superiores (5GHz, 6GHz) no son compatibles.

Windows (PowerShell)

Windows (PowerShell)

Ready-to-use Scripts

Scripts Listos para Usar

Copy these commands into a .ps1 file or run them directly in PowerShell.

Copie estos comandos en un archivo .ps1 o ejecútelos directamente en PowerShell.

BioLighting - Get Status / Obtener Estado

Invoke-RestMethod -Uri "http://192.168.1.50/api/light" -Method Get

BioLighting - Set Color / Configurar Color

$body = @{r=255; g=180; b=100; intensity=100} | ConvertTo-Json
Invoke-RestMethod -Uri "http://192.168.1.50/api/light" -Method Post -Body $body -ContentType "application/json"

BioShaker - Get Status / Obtener Estado

Invoke-RestMethod -Uri "http://192.168.1.51/status" -Method Get

BioShaker - Set RPM / Configurar RPM

Invoke-RestMethod -Uri "http://192.168.1.51/rpm?value=120" -Method Get

Scheduling (Task Scheduler)

Programación (Programador de Tareas)

  1. Save the command in a file, e.g., script.ps1. Guarde el comando en un archivo, ej. script.ps1.
  2. Open "Task Scheduler" and create a Basic Task. Abra "Programador de Tareas" y cree una Tarea Básica.
  3. Action: Start a Program. Program: powershell.exe. Acción: Iniciar un programa. Programa: powershell.exe.
  4. Arguments: -ExecutionPolicy Bypass -File "C:\Path\To\script.ps1". Argumentos: -ExecutionPolicy Bypass -File "C:\Ruta\A\script.ps1".

macOS (Terminal/cURL)

macOS (Terminal/cURL)

Ready-to-use Scripts

Scripts Listos para Usar

Use these commands in Terminal or save them as .sh files.

Use estos comandos en Terminal o guárdelos como archivos .sh.

BioLighting - Get Status / Obtener Estado

curl "http://192.168.1.50/api/light"

BioLighting - Set Color / Configurar Color

curl -X POST "http://192.168.1.50/api/light" \
     -H "Content-Type: application/json" \
     -d '{"r":255,"g":180,"b":100,"intensity":100}'

BioShaker - Get Status / Obtener Estado

curl "http://192.168.1.51/status"

BioShaker - Set RPM / Configurar RPM

curl "http://192.168.1.51/rpm?value=120"

Scheduling (Automator)

Programación (Automator)

  1. Open "Automator" and create a "Calendar Alarm". Abra "Automator" y cree una "Alarma de Calendario".
  2. Drag "Run Shell Script" from the actions library. Arrastre "Ejecutar script de shell" desde la biblioteca de acciones.
  3. Paste one of the cURL commands above. Pegue uno de los comandos cURL de arriba.
  4. Save. Calendar will open; set the event time and repeat interval. Guardar. Se abrirá Calendario; establezca la hora del evento y el intervalo de repetición.
